Animation Utility Classes

These animation classes can be applied to any element to add interactive effects:

Hover Animations

hover-lift

Lifts the element on hover

<div class="hover-lift">...</div>
hover-shadow

Increases shadow on hover

<div class="hover-shadow">...</div>
hover-scale

Scales the element on hover

<div class="hover-scale">...</div>

Entrance Animations

Add these classes to elements that should animate when they appear in the viewport:

.animate-fade-in
.animate-fade-in-up
.animate-slide-in-left
.animate-slide-in-right
